# PROJECT METODE UJI
Project ini mengenai klasifikasi biji gandum dari beberapa variabel agar dapt memprediksi type class dari biji gandum tersebut
## Variabel yang digunakan
- Area
- Perimeter
- Compactness
- Length of Kernel
- Width of Kernel
- Asymmetry Coefficient
- Length of Kernel Groove
